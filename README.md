# CS527 project
A types dependencies visualization system for STARTS

## Development

  ### Environment Requirement 
  - Python 2.7
  - pip
  
  ### Install **flask**
  
  ```bash
  $ pip install -U flask
  ```
  ### Set up Web Server

  ```bash
  $ python run.py
  ```
