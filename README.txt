Instruction:

1. Install

a. Install Python 2.7
b. Install pip
c. Install Flask Framework: open terminal and execute "pip install -U flask"
d. Install scikit-learn: open terminal and execute "pip install -U scikit-learn"

2. Run the program:

a. Clone the folder "Class_dependencies_vis"
b. On this folder, open a terminal and run "python run.py"
c. In your browser, visit "localhost: 5000" or "127.0.0.1: 5000"

3. File description:
Class_dependencies_vis: the project
commons_fileupload: a STARTS project that provide class dependencies file
progress: screenshots of the project
